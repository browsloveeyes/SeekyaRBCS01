测试前先在浏览器输入网址http://168.2.5.26:1906/services/WSInterface?wsdl看能否打开
测试流程：
1.点开“配置”,选择合适的串口号，确定
2.在“配置”窗口的“扫描枪支持”栏下的URL网址框输入“http://168.2.5.26:1906/services/WSInterface?wsdl”，确定，关闭
3.在“配置”窗口的“模板选择”栏下选择打印模板（一般为C:\Program Files (x86)\SeekyaRBC 1.3.6\SeekyaRBC 1.3.6\Data\template\template.xlsx）,关闭
4.在“检验条形码”框输入患者的**病人id**，确定（观察是否加载患者信息）

5.点击测试
